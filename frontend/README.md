## Como rodar o projeto baixado

### Para instalar todas as dependências indicadas pelo package.json
```bash
npm install
```

### Para rodar o projeto React
```bash
npm run dev
```

### Acessar no navegador
```bash
http://localhost:3000
```

### Para instalar as dependências de ícones
```bash
npm install react-icons
```
<hr>

### Para criar um projeto React
```bash
npx create-next-app
```

### Acessar o diretório do projeto
```bash
cd [nome-da-pasta]
```

### Para rodar o projeto React
```bash
npm run dev
```
